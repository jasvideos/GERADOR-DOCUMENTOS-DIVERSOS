# Gerador de Documentos Diversos

Aplicativo web desenvolvido em React para geração de documentos em PDF, como recibos, contratos, declarações e currículos.

## Funcionalidades

- Geração de PDFs utilizando `jspdf`.
- Visualização em tempo real do documento.
- Diversos modelos disponíveis (Recibos, Contratos, CV, etc.).
- Interface amigável e responsiva.

## Como rodar o projeto

1.  Instale as dependências:
    ```bash
    npm install
    ```

2.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

3.  Acesse no navegador (geralmente em `http://localhost:5173`).

## Tecnologias

- React
- Vite
- jsPDF
